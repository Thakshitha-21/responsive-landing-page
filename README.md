# TechSolutions - Responsive Landing Page

A modern, fully responsive landing page built with HTML, CSS, and JavaScript for Week 1 task submission.

## Features

- **Fully Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Modern UI/UX** - Clean and professional design with beautiful gradients
- **Mobile-First Approach** - Optimized for mobile users
- **Smooth Animations** - CSS animations and transitions for better user experience
- **Cross-Browser Compatible** - Works on all modern browsers
- **SEO Friendly** - Semantic HTML structure
- **Accessibility** - Proper focus management and keyboard navigation

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Flexbox, Media Queries, Animations
- **JavaScript** - Mobile menu functionality and interactions
- **Font Awesome** - Icons
- **Google Fonts** - Typography
